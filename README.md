#  Air Quality Monitoring and Analysis System (AQMAS)

##  Description

This project is a Python-based system that monitors and analyzes air quality. It generates environmental data such as PM2.5, PM10, temperature, and humidity, calculates the Air Quality Index (AQI), and provides alerts based on pollution levels.

## Features

* AQI Calculation
* Air Quality Classification
* Alert System
* Data Storage using CSV
* Data Visualization using graphs

##  Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib

##  How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the program:
   python main.py

##  Output

* Displays air quality report
* Generates graphs for AQI and pollution levels
* Saves data in CSV file
